# 💰 Finance Manager

## 🌐 Project Overview

**Finance Manager** is a simple and modern web-based application that helps users manage their personal finances.  
The app allows users to track their **income** and **expenses**, view **statistics**, and manage **categories**.  

All operations are stored locally in the browser using `LocalStorage`, ensuring the app runs smoothly without a backend server.  
It is lightweight, fast, and designed with a beautiful modern interface.

🔗 **Live Website:** [Finance Manager Live](https://mohamedms-max.github.io/finance-manager2/)

---

## 🚀 Features

- 🔒 **User Authentication** — Create an account and securely log in.  
- 💰 **Add Transactions** — Record your income and expenses easily.  
- 📋 **View All Transactions** — See all operations in a clear table format.  
- 📊 **Auto Calculations** — Instantly view total income, expenses, and balance.  
- 🗂️ **Category Management** — Add and delete financial categories.  
- 🎨 **Modern UI** — Responsive design with stylish finance-themed backgrounds.  
- 🛡️ **Security** — All pages are protected; access requires login.  
- ⚙️ **LocalStorage System** — Data is saved locally on the user’s device.

---

## 🧭 Page Flow

1️⃣ **Landing Page (`landing.html`)** → Welcome screen  
2️⃣ **Login (`login.html`)** → User authentication  
3️⃣ **Signup (`signup.html`)** → Account creation  
4️⃣ **Home (`index.html`)** → Welcome dashboard  
5️⃣ **Add Operation (`add.html`)** → Add income or expense  
6️⃣ **Operations List (`list.html`)** → View all records  
7️⃣ **Statistics (`stats.html`)** → Summary of finances  
8️⃣ **Categories (`categories.html`)** → Manage custom categories  

Each page is connected and protected. The user cannot access inner pages without logging in.
